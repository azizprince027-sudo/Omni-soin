## 📋 Description
**OmniSoin** est une solution numérique conçue pour moderniser l'accès aux soins de santé. La plateforme centralise trois services critiques en une interface unique, fluide et intuitive, permettant de réduire drastiquement les délais d'attente et de simplifier le parcours patient.

### Objectifs clés :
*   **Prise de rendez-vous :** Digitalisation complète de la prise de contact avec les médecins spécialistes.
*   **Conseils Santé :** Accès à une base de connaissances pour une meilleure hygiène de vie.
*   **Géolocalisation :** Identification rapide des pharmacies de proximité pour les urgences.

---

## 🚀 Fonctionnalités Clés
*   **Système d'Inscription Avancé :** Formulaire intelligent avec gestion du *Drag et Drop* pour les documents officiels (CNI, CMU).
*   **Interface Responsive :** Design adaptatif optimisé pour une utilisation sur mobile et desktop.
*   **Validation Dynamique :** Utilisation de JavaScript pour sécuriser et valider les entrées utilisateurs en temps réel.
*   **Architecture Sémantique :** Code HTML5 structuré pour un référencement (SEO) et une accessibilité optimisés.

---

## 🛠 Stack Technique
Le projet repose sur une architecture front-end robuste et légère pour garantir des performances maximales :

*   **Structure :** [HTML5](https://developer.mozilla.org/fr/docs/Web/HTML) (Sémantique avancée).
*   **Style :** [CSS3](https://developer.mozilla.org/fr/docs/Web/CSS) (Flexbox, CSS Grid, Design System personnalisé).
*   **Intelligence :** [JavaScript ](https://developer.mozilla.org/fr/docs/Web/JavaScript) (Manipulation du DOM, gestion d'événements, validation de fichiers).
*   **Ressources externes :**
    *   **Iconographie :** FontAwesome.
    *   **Typographie :** Google Fonts (Poppins).
